Sobre o curso de Análise e Desenvolvimento de Sistemas

O curso de Análise e Desenvolvimento de Sistemas forma profissionais capazes de desenvolver, implementar e manter sistemas de software, com uma abordagem focada em programação, bancos de dados e desenvolvimento de soluções tecnológicas.

Sobre o curso de Ciência de Dados

O curso de Ciência de Dados capacita o aluno a trabalhar com grandes volumes de dados, utilizando técnicas de análise estatística, machine learning e programação para transformar dados em informações estratégicas para empresas e organizações.
